# 🐾 电子桌宠 - 谛听

> 在桌面上记录你的日常，陪伴你，提醒你喝水吃药。

## 🎯 项目简介
**谛听** 是一款桌面电子宠物助手，旨在解决社恐人群的孤单感，为他们提供温暖的陪伴；同时也适用于孤寡老人，代替子女进行日常关怀与提醒。

## ✨ 核心功能
- **💬 日常陪伴**：作为实体电子宠物，时刻陪伴在用户身边，缓解孤独。
- **⏰ 智能提醒**：定时提醒用户喝水、吃药等重要日常行为。
- **📝 生活记录**：自动记录用户的日常点滴，形成专属的生活记忆。
- **🤖 AI 驱动**：基于 AI 技术，提供更自然、更懂你的交互体验。

## 👥 适用人群
- **社恐人群**：需要无声但温暖的陪伴，减少社交压力。
